# UmaCheck (ウマチェック)

ウマ娘のサークルメンバーのファン数リストを自動でキャプチャし、Gemini AIで数値を抽出する Windows デスクトップアプリです。

> 制作者: 선두의경치 (韓国サーバー) · v0.1.1

**他の言語:** [한국어](README.md) · [English](README_EN.md) · [繁體中文](README_ZH.md)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

---

## 主な機能

### キャプチャ
- 起動中のウマ娘ゲームウィンドウを自動検出
- サークル情報画面を自動スクロールしながらスクリーンショットを収集
- 重複部分を自動除去し、1枚の縦長画像に合成
- 結果を `output/` フォルダに PNG として保存

### 分析
- Gemini API を使い、キャプチャ画像からファン数を自動抽出
- 対応モデル: `gemini-2.5-flash` (推奨)、`gemini-2.0-flash`、`gemini-2.0-flash-lite`、`gemini-2.5-pro`
- 連続する重複数値の自動除去オプション
- 結果をクリップボードにコピー可能
- 画像のドラッグ＆ドロップまたはファイル選択に対応

### 設定
- ゲームインストールフォルダのパスを設定
- スクロール間隔・スクロール量・類似度しきい値を調整

### その他
- アプリからゲームを直接起動可能
- 多言語対応: 한국어 / English / 日本語 / 繁體中文

## 動作環境

- **OS:** Windows 10 / 11
- **Python:** 3.12 以上
- **ゲーム:** ウマ娘 プリティーダービー (KakaoGames 韓国サーバー)
- **Gemini API Key:** [Google AI Studio](https://aistudio.google.com/apikey) で無料取得

```bash

```

## 使い方

1. 初回起動時に**初期設定モーダル**が表示されます。
   - ゲームインストールフォルダ（`UMMS_Launcher.exe` があるパス）を入力します。
   - Gemini API Key を入力します。

2. **キャプチャ**タブに移動します。
   - ウマ娘を起動し、**サークルメニュー → サークル情報**画面へ移動します。
   - ファン数リストを**一番上**までスクロールした状態で **[キャプチャ開始]** を押します。
   - アプリが自動でスクロール・キャプチャし、完了後に画像を合成します。

3. **分析**タブに移動します。
   - キャプチャ完了後に自動で移動するか、手動で画像をドラッグ＆ドロップします。
   - Gemini モデルを選択し、**[Extract]** を押します。
   - 抽出されたファン数リストを確認し、クリップボードにコピーします。

## 設定項目

| 項目 | デフォルト | 説明 |
|------|-----------|------|
| ゲームフォルダ | `C:\kakaogames\umamusume` | UMMS_Launcher.exe があるフォルダ |
| スクロール間隔 | `1.0` 秒 | スクロール後、次のキャプチャまでの待機時間（低スペック PC は高く設定） |
| スクロール強度 | `6` | 1回あたりのスクロール量 |
| 類似度しきい値 | `99.5%` | この値以上の類似度でスクロール終端と判定 |

## 注意事項

- **Windows 専用**アプリです。macOS・Linux では動作しません。
- キャプチャ中はマウスを操作しないでください。アプリが自動制御します。
- Gemini API Key は `config.json` に保存されます。このファイルを外部に公開しないでください。
- Gemini API の無料枠には使用量制限があります。大きな画像の分析には `gemini-2.5-flash` を推奨します。

## プロジェクト構成

```
uma-fancheck-app/
├── main.py          # アプリのエントリポイント、pywebview API 定義
├── config.json      # ユーザー設定（自動生成）
├── requirements.txt # 依存ライブラリ一覧
├── icon.png         # アプリアイコン
├── src/
│   ├── capture.py   # ゲームウィンドウ検出・自動キャプチャ
│   └── stitch.py    # 重複除去・画像合成
├── web/
│   └── app.html     # アプリ UI (HTML/CSS/JS)
└── output/          # キャプチャ結果の保存フォルダ
```
